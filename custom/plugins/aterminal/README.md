## Install
> :warning: You need to install [Oh My ZSH](https://github.com/robbyrussell/oh-my-zsh)

#### Run installer
`bash < <(curl -s -S -L https://raw.githubusercontent.com/guiferpa/aterminal/master/installer.sh)`

#### Add aterminal plugin
`vim $HOME/.zshrc`
